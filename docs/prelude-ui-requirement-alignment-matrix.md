# Prelude UI Requirement Alignment Matrix

## 使用边界

- 本矩阵记录网页端已完成确认的 Q-001 到 Q-137，以及 Q-138 额外加载动画需求。
- Q-001 到 Q-137 是逐组件/规则确认项。
- Q-138 是额外需求：引入 Math Curve Loaders 的 Rose Three 加载动画，并根据 Prelude 主题调整样式。
- 本地 agent 必须直接按本矩阵和 `prelude-ui-style-convergence-plan.md` 执行，不得二次询问已确认项。
- “保持”表示保持当前产品方向，但仍必须补齐 token、可访问性、focus、Tooltip、数据源、静态扫描和验证要求。
- 只有当矩阵与当前代码事实冲突、出现未覆盖的新组件/新范围、执行会破坏已确认规则、或验证证明计划不可落地时，本地 agent 才暂停报告。
