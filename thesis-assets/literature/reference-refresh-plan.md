# 参考文献翻新计划

1. 翻新目标：
   - 正式参考文献 25±5 条；
   - 中文 10-14 条；
   - 英文 10-14 条；
   - 官方技术文档 4-6 条。

2. 质量规则：
   - A / A- / B+ 优先；
   - B- 严格限用；
   - C 类不进入正式参考文献；
   - 不完整元数据不进入正式参考文献；
   - 所有 DOI、期刊页、出版社页必须人工核验。

3. 来源优先级：
   - 中文核心/高质量综述；
   - 高质量英文期刊/会议；
   - arXiv 仅作前沿补充；
   - 官方技术文档仅作技术依据；
   - 普通网页不得作为核心研究文献。

4. 输出文件：
   - reference-refresh-2026-06.md；
   - candidates/；
   - 更新版 quality-review 草案；
   - 更新版 evidence-map 草案。

5. 禁止事项：
   - 不直接改正文；
   - 不直接改正文编号；
   - 不直接生成最终参考文献列表；
   - 不引用低质量文献。
