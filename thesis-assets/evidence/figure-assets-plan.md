# 图表资产与 nature-figure 使用计划

1. Mermaid / draw.io / nature-figure 分工：
   - Mermaid：事实图源；
   - draw.io：人工排版微调；
   - nature-figure：数据图表、科研图表、可视化资产增强。

2. nature-figure 必须参与的候选场景：
   - 测试数据图；
   - 构建耗时对比图；
   - Demo/MIMO 延迟对比图；
   - 已测 / 未测状态矩阵；
   - 答辩中需要的科研式数据图。

3. nature-figure 禁止事项：
   - 不得替代代码事实判断；
   - 不得生成营销图；
   - 不得绘制脱离源码的架构节点；
   - 不得直接修改正文引用。

4. 图表进入论文条件：
   - 有数据源；
   - 有图表登记；
   - 有用户与审查官复核；
   - 有正文引用位置规划。
