### 2.3 Docker Compose BYOK 真实 LLM 功能链路记录

- 命令范围：`docker compose config --quiet`、`docker compose up -d --build mysql redis rabbitmq backend`、HTTP API 注册/登录/创建会话/`finish`、RabbitMQ 管理接口、MySQL 查询。
- 执行环境：本机 Windows 11 + Docker Compose 容器环境。
- 运行口径：关闭 dev fixture；通过用户级 BYOK 输入 OpenAI-compatible endpoint 与 API Key，调用模型发现接口后选择运行模型并保存。API Key 不写入仓库；具体运行模型不作为论文或配置默认值。
