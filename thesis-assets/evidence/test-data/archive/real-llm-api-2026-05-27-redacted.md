## 伴随性 Bug 修复记录

测试过程中修复了阶段推进校验问题：原逻辑只检查最后一条消息是否为 `assistant`，导致真实流式追问落库后用户难以推进阶段。修复后，逻辑仅在当前阶段存在 `assistant` 发问且没有任何 `user` 回复时，才判定存在挂起提问。
